The iX Cube scene is a simple example of an XML3D scene. As in all examples on this site, basic controls are provided. The scene can be rotated by holding the left mouse key and moving the mouse. Similarly, this example can be zoomed in or out by pressing the right mouse key and moving the mouse back and forth.

Diese einfache XML3D-Szene wurde im aktuellen iX-Artikel erklärt. Wie in allen hier gezeigten Beispielen, kann durch die Szene navigiert werden. Die Szene kann rotiert werden, indem der linke Mausknopf gedrückt und die Maus bewegt wird. Durch das Drücken der rechten Maustaste und einer gleichzeitigen Vorwärts- bzw. Rückwärtsbewegung kann man rein- oder rauszoomen.
